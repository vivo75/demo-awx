Role Name
=========

A role that installs memcached

Role Variables
--------------

The most important variables are listed below:

``` yaml
mc_port: 11211
mc_user: memcached
mc_maxconn: 1024
mc_cachesize: 256
mc_options: ""
mc_ipaddress: 127.0.0.1
```
